# aceJump快捷键

## ctrl + ; 不是插件自带，我改过键
进入word模式，本屏幕里面所有的单词都会标记出来

## ctrl + shift + ; 插件自带的，没有改过键
进入行模式，将所有的行标记出来

## ctrl + command + ; 不是插件自带，我改过键
进入文字模式，不仅仅局限与单词，可以选择任意一个字符，但是性能消耗较大，而且比较慢

## ctrl + option + ; 不是插件自带，我改过键
进入target模式，就是选择该变量之后，自动进入到该变量的定义里面去，看代码超级好用
